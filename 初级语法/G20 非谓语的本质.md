## 1. 产生谓语的条件
1. S[^1] + V[^2]
2. S + V + And / or / but…+ V
3. S + V + 连接词（并列 / 从属连词） + S + V 

## 2. 英语的第一原则：`一山不容二虎，除非一公一母`
1. SV，非V
2. 非V，SV
3. S  非V  V

## 3. 非谓语的本质
|   名称   |   抽象   |   本质    |        运用        |
|:--------:|:--------:|:---------:|:------------------:|
|  不定式  |  to do   |  不确定   | 将来、目的、一次性 |
|  动名词  |  doing   |   确定    |  长期、稳定、习惯  |
| 现在分词 |  doing   | 分身/省略 | 与谓语动词同步发生 |
| 过去分词 | done(vi) |           |        完成        |
|          | done(vt) |           |     完成、被动     |









[^1]: 主语
[^2]: 谓语
