1. `完整`：主句完整
2. `抽象`：主句中有==抽象n==
3. `等同`：从句 = 抽象n

## 同位语从句 vs 定语从句
- 相同点：==主句完整==
|        同位从         |               定从               |
|:---------------------:|:--------------------------------:|
|       抽象名词        |        抽象+具体（笼统）         |
| 等同==（从句=抽象）== |              不等同              |
|     that不作成分      | that（关代【n】） 【主、宾、表】 |
|                       |                                  |
