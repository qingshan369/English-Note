## 比较状语从句
### than、as…as、the + 比较级，the比较级
|                         |                    |
|:-----------------------:|:------------------:|
|          than           | 越级比较：有差异性 |
|          as…as          |      同级比较      | 
| the + 比较级，the比较级 |    倚变比较，渐进式比较               |
#### as…as的考点
1. 中间是adj / dav原级
2. 倍数放前面
