## 定从的主谓一致
- “one of+复数名词”位于关系代词前作先行词时，关系代词在从句中作主语，从句的动词通常用复数，但当one前有the,the only,the very等修饰时，从句的谓语动词要用单数。

## 非限定从中as与which
- 相同点：代唯一名词，代整个主句
- 不同点：
|              |                                  |                          |
|:------------:|:--------------------------------:|:------------------------:|
|     意思     |       which表达的是这件事        |         as是正如         |
|     位置     |         which位于主句后          |          as任意          |
| 时间（本质） | which分先后 主句先发生，从句评价 | as不分先后，把主句当事实 |
